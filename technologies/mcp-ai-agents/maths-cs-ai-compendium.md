## Đây là gì?

**Maths, CS & AI Compendium** là **sách giáo khoa mở** (intuition-first): toán → ML → NLP/CV/speech/multimodal → hệ thống / GPU / inference / ML design — 18 chương, không chỉ học để thi. Có **MCP server** để Claude Code / Cursor đọc compendium như knowledge base (cần clone local).

Khác [Easy-Vibe](easy-vibe.md) (vibe coding ship app) và [Learn Claude Code](learn-claude-code.md) (harness): đây = **nền lý thuyết AI/ML + MCP**. Gần [Train LLM From Scratch](../devtools/train-llm-from-scratch.md) (thực hành train) nhưng rộng hơn (toán + systems).

**Cùng kiểu:** [Easy-Vibe](easy-vibe.md), [Learn Claude Code](learn-claude-code.md), [AI Engineering Coach](ai-engineering-coach.md), [Train LLM From Scratch](../devtools/train-llm-from-scratch.md), [prompts.chat](prompts-chat.md).
