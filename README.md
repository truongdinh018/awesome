## Map catalog

```
repos/          → ai đã star (#, Primary, Tags)
technologies/   → bài chi tiết theo domain
categories/     → subgroup & use-case
TAGS.md         → tìm theo capability
```

```
awesome-ai/
├── README.md              ← bạn đang ở đây
├── TAGS.md
├── CHANGELOG.md
├── categories/            ← taxonomy
├── repos/                 ← index star
└── technologies/
    ├── mcp-ai-agents/ … security/   ← 7 domain
    ├── bookmarks/
    └── cli-anything/
```

## Thêm ★ mới

1. Star repo → dòng mới trong [repos/](repos/README.md)  
2. Viết `technologies/<domain>/<slug>.md` theo [WRITING.md](technologies/WRITING.md)  
3. Ghép subgroup trong [categories/](categories/README.md)  
4. Cập nhật [TAGS.md](TAGS.md) nếu cần  
5. Bump số ★ trên file này + dòng đầu [CHANGELOG.md](CHANGELOG.md)

Harness CLI-Anything → `technologies/cli-anything/<slug>.md`
